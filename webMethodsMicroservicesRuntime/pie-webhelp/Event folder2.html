Package and Description

WmPublic. Creates a subscription for a specified event.

WmPublic. Document type for the name of the service in the invocation path when an exception occurred and the index that indicates the step at which the exception occurred.

WmPublic. Removes an event handler from the subscription list for a specified event.

WmPublic. Deprecated - There is no replacement document type. Document type that defines the structure of an event document.

WmPublic. Deprecated - There is no replacement service. Converts an EDA event in the form of an XML string to a document (IData object).

WmPublic. Deprecated - There is no replacement schema. Schema that defines the structure and data types used for the event header in the pub.event.eda:event document type.
