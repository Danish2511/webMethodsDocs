Overview of Creating a GraphQL Descriptor
Creating a GraphQL descriptor is a process that involves the following basic stages.
Stage 1
Create a new GraphQL descriptor on Integration Server.
During this stage, you use the Service Development perspective to create a new GraphQL descriptor based on the GraphQL schema.
Stage 2
Build resolver services for the GraphQL descriptor.
During this stage, you specify the logic to build the resolver services.
Stage 3
Specify properties for the GraphQL descriptor.
During this stage, you specify the properties for the GraphQL descriptor in the Properties view.
