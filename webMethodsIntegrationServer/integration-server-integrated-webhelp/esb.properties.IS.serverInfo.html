Server Information
View general information about a server from the Server Information page. To open this page, select File > Properties > Server Information.
Property
Description
Type
Type of server. This will be Integration Server.
Name
Name assigned to the Integration Server.
Location
Host name and port address of the Integration Server
User Name
The user name you use to connect to this Integration Server.
Connected
Indicates whether or not you are currently connected to this Integration Server.
VCS Enabled
Indicates whether this Integration Server is configured to use a version control system (VCS). This property has a value of true only when the WmVCS pakcage is installed on Integration Server.
Proxy
Specifies the proxy server as set on Window > Preferences > General > Network Connections.
