Viewing Notifications
To view a notification
1. Open Integration Server Administrator.
2. Click . Notifications are displayed as a list. Hover over a notification to see the complete information.
Each notification has an icon that represents the severity of the event, a subject, time of the notification, notification type, and a description. A description with multiple lines of text is displayed as a single line ending with ellipses, which indicates that the remaining lines are displayed on hover. If there are multiple unread notifications, scroll to the required notification to read it.
