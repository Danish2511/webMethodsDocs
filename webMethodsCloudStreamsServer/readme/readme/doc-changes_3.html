Release 10.15
In the webMethods CloudStreams Connector Concepts Guide, the Managing Connector Listeners chapter now contains four new sections, namely Viewing Connector Listener Runtime Status, Understanding Connector Listener Status, Understanding Connector Listener State Transition, and Handling Errors. The Viewing Connector Listener Runtime Status section describes how to view the runtime status of a configured connector listener by using the Integration Server Administrator. The Understanding Connector Listener Status section describes the behavior of connector listener statuses in Integration Server Administrator. The Understanding Connector Listener State Transition section describes how the status of a configured connector listener is determined and transitioned in Integration Server Administrator. The Handling Errors section lists the most common error codes that users may encounter, explanation of the cause, and potential resolutions.
In the webMethods CloudStreams API Reference Guide, the Product Setup chapter now contains three new sections, namely Metadata APIs, Asset Management APIs, and Runtime APIs. The Metadata APIs section explains all Metadata APIs that can be used to query connectors and CloudStreams server metadata. The Asset Management APIs section explains all Asset APIs that can be used to perform CRUD (create, read/retrieve, update, delete) operations on CloudStreams assets. The Runtime APIs explains the Runtime APIs that can be used to execute a service and retrieve the current status of a connection.
The Advanced Settings chapter now contains a new section wst.listener.events. that explains the parameters to be configured in the CloudStreams properties file if you want to use a database to store event data.
The Cloud Connections, Services, and Connector Listeners chapter now contains a new section named Generating Azure Storage SAS Tokens while configuring connections that describes how to generate shared access signature (SAS) tokens while configuring a connection to Microsoft Azure Storage, from the connection configuration page in webMethods Integration Server Administrator.
From this release, the CloudStreams Analytics feature is deprecated. The chapter is updated to include the Deprecation announcement.
