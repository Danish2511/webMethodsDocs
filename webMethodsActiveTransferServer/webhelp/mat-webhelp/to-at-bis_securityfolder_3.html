wm.mft.security.pgp:generatePGPKeyFiles
Generates an OpenPGP key pair (public and private key) that can be used to configure PGP in ActiveTransfer Server. The key pair is generated with the following properties:
Property
Description
keySize
2048
Supported Compression Algorithms
ZIP, ZLIP, Uncompressed
Supported Hashing Algorithms
SHA1, SHA256, SHA384, SHA512, MDS
Supported Symmetric Algorithms
CAST5, AES_128, AES_192, AES_256, TWOFISH
Key Common Name
commonName
Input Parameters
privateKey Path
String Specifies the directory in the server where the key files are generated. The files are generated with the names “private.key” and “private.pub” in the directory.
validityDays
String Optional. Specifies the number of days for which the keys will be valid. The default value is 365.
Password
String Optional. Specifies the password for the private key.
Output Parameters
Document type
String Contains “success = true” if the service runs successfully. Contains an error message if the service fails.
