wm.mft.assets.rec.event:ScheduleCriteria
Document type that contains the schedule frequency and details for an ActiveTransfer Server scheduled event.
Variables
scheduleFrequency
String (Optional) Specifies the frequency with which ActiveTransfer Server should run the schedule event configured. The schedule frequency options are as follows:
Daily (default)
Hourly
Run Once
Monthly
Manual
Fixed Interval
Yearly
Weekly
scheduleDetails
Object Contains one of the document types in the wm.mft.assets.rec.event.scheduleData folder and is dependent on the scheduleFrequency selected for the scheduled event. The object can take one of the following values:
daily
hourly
runOnce
monthly
manual
fixedInterval
yearly
weekly
