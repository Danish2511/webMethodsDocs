Searching Using Keywords (Simple Search)
To search using keywords
1. In My webMethods: Administration > Integration > B2B > Partner Administration > asset to search for .
2. Click the Keyword tab and in the text box, perform the actions listed in the table:
Type of Keyword Search
Action to Perform
Exists in Trading Networks that you are authorized to view
Leave it blank.
Contains one or two keywords
Type one keyword or two keywords separated by a space.
Contains a specified string
Type a string enclosed in double quotes.
3. Click Go. Trading Networks searches are listed in the table, as follows:
Asset Type
Trading Networks Searches For
Profiles
The profile external IDs and corporation name.
If you type one keyword or a string, it must be either an external ID or the corporation name.
If you type two keywords, one must be an external ID and the other must be the corporation name.
TPAs
The TPA agreement IDs, sender name, and receiver name. The keyword or a string you type must be either an agreement ID, sender name, or the receiver name.
Other assets
The asset names.
