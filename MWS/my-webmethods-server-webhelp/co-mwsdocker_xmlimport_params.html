Parametrizing Properties in xmlImport Files
In addition to supplying custom values to the instance and cluster configuration files for My webMethods Server containers, you can also parametrize properties in the xmlImport files that you use to create server resources. Custom values can be supplied either as environment variables or through system properties in the custom_wrapper.conf file for My webMethods Server.
When you start a container, My webMethods Server substitutes all values you provide from the Docker CLI or in the custom_wrapper.conf file before processing the xmlImport files.
