About Creating New Workspaces
 
Creating New Empty Workspaces
Using “Save As” to Create a New Workspace
When creating a new workspace, you can either create:
An empty workspace
A workspace that is a copy of an existing workspace
To copy an existing workspace, you use “save as” functionality to save an existing workspace using a new name. As you make changes to the new and original workspaces, My webMethods does not synchronize the changes. The two workspaces are distinct.
